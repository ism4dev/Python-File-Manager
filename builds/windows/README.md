## Finalidade do SHA256

O SHA256 é uma **assinatura única do arquivo** que permite verificar sua integridade e garantir que o arquivo **não foi alterado ou corrompido** após o download.

Para verificar no Windows:

```powershell
Get-FileHash "Python File Manager.exe" -Algorithm SHA256
````

Compare o hash gerado com o hash oficial abaixo:

**SHA256 oficial:** 796a340cb3c78c06afc3d57adef6199213713b3973dd5262ace521a19262939b

Se os hashes coincidirem, o arquivo está íntegro e seguro para uso.
